Price File Comparison Tool

1. Save a current/old price file from excel with the following columns in "text (tab delimited) (*.txt)" format.

Example:

Vendor Item NumberVendor Price
A12315.00
23419.99
C34530.00

2. Save a new price file from excel with the following columns in "text (tab delimited) (*.txt)" format.

Example:

Vendor Item NumberVendor Model NumberVendor Price
A12312315.99
23422.50
C34525.00

3. Click the Browse... button and select the current/old price text file

4. Click the Browse... button and select the new price text file

5. Press the Load Files button to process the Old and New price files.

6. Press the Export Report button and save the file. (alternatively: you can select "open" instead of saving the file to open a temporary file instead of saving to your hard drive.)

7. Create a new Microsoft Excel document and format column A as "text" and column E as a percentage.

8. Open the report generated in step 6 with notepad or an equivalent text file editor

9. Copy it's entire contents, select the cell A1 in the excel document, and paste the data. (the keyboard shortcut ctrl + A will highlight everything in the file. ctrl + C will copy the content that is highlighted. ctrl + V will paste the data).
